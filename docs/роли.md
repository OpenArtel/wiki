# Роли

Зоны ответственности не прибиты гвоздями, долевое владение подталкивает к включению и выполнения задач из смежных зон.

Ключевой аспект классных команд в том, что люди делают не свою работу. В любой цепочке задач есть "бутылочное горлышко" и подход с долями позволяет помочь разгрузить его.

Org Ops (Organizational Operations) - отвечает за структуру, доли, процессы, справедливость, формируют стратегию, финансы, контракты.

Финансы / Ops — учёт, упралвение долевого фонда, выплаты.

HR / People Ops — рекрутинг, онбординг, конфликты.

Маркетинг — привлечение лидов, создание контента, поддержка имиджа и анализ эффективности.

Продажи / BD (Business Development) — продажи, сопровождение клиентов и развитие через партнёрства и нетворкинг.

Project Manager / Prod Owner — приоритизация, Sprints, связь с клиентом, приёмка.

Tech Lead / Архитектор — архитектура, code-review, релизы, беклог.

Разработчик (frontend/backend) — реализация фич, тесты, поддержка.

Дизайнер / UX/UX — макеты, визуальный стиль и айдентика продукта.

QA / Release Engineer — тестирование, стабильность релизов и поддержка хотфиксов.
