# Разработка

## Оглавление

- Получена инициатива
- Планирование
- Разработка (основной цикл)
- Релиз
- Поддержка и улучшение

## Получена инициатива

- Инициатива: идея, заказ, запрос рынка.
- Сбор требований.
— Формулировка гипотез.
— Оценка трудоёмкости и стоимости.

## Планирование

— Формирование команды.
— Подготовка backlog и дорожной карты.

## Разработка (основной цикл)

— Постановка задач.
— Кодирование / проектирование / прототипирование.
— Тестирование.
— Сдача инкремента.

## Релиз

— Документация.
— Развёртывание / поставка.
— Коммуникация с заинтересованными сторонами.

## Поддержка и улучшение

— Сбор обратной связи.
— Анализ метрик.
— Постановка улучшений в backlog.
